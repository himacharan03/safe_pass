# SafePass – V2V Overtake & Collision Prevention System

A Python Flask web application that simulates Vehicle-to-Vehicle (V2V) communication, overtaking decision-making, and collision prediction. 

## Features
* V2V communication simulation
* Rule-based overtaking decision engine
* Real-time collision prediction graph
* Live vehicle animation on a modern dark theme UI
* 3 vehicle overtaking scenario simulation

## Installation

```bash
pip install flask matplotlib numpy
```

## Running the System

```bash
python app.py
```

Open `http://127.0.0.1:5000` in your browser.
